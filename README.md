
# Student Authentication System

This is a Student Authentication System developed using PHP and MySQL as part of my PHP & MySQL Internship at ApexPlanet Software Pvt. Ltd.

The project allows users to register and log in securely. It includes role-based authentication where administrators can manage student records and students can view the records.

## Features

- User Registration
- User Login
- Admin and Student Roles
- Add, View, Edit and Delete Student Records
- Search Functionality
- Pagination
- Prepared Statements
- Client-side Validation
- Server-side Validation
- Session Management
- Logout and Delete Confirmation

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## How to Run

1. Install XAMPP and start Apache and MySQL.
2. Copy the project folder into the htdocs folder.
3. Import the database into phpMyAdmin.
4. Open the project in your browser.


http://localhost/student_auth-advanced/login.php
